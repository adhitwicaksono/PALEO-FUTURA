# Project Identity

## Name

**PALEO-FUTURA**

## Full concept

**AI-assisted evolutionary genomics framework for ancestral inference and future evolutionary forecasting**

## Short description

PALEO-FUTURA connects ancestral evolutionary inference with scenario-based future evolutionary forecasting to generate testable biological hypotheses.

## One-line description

**A scientific time machine with error bars for evolutionary genomics.**

## Tagline

**PALEO asks what evolution left behind; FUTURA asks what evolution may do next.**

---

## Scientific positioning

PALEO-FUTURA is best described as:

- a research-software framework
- an evolutionary genomics reasoning layer
- an AI-assisted hypothesis-prioritization system
- an uncertainty-aware interpretation platform

It should not be described as:

- an oracle
- a resurrection engine
- a deterministic future predictor
- a replacement for experimental biology
- a fully automated evolutionary simulator

---

## Core modules

| Module | Direction | Role |
|---|---:|---|
| **PALEO** | Past | ancestral inference and evolutionary archaeology |
| **FUTURA** | Future | scenario-based evolutionary forecasting |
| **Evidence Matrix** | Present | structured gene-level and pathway-level evidence |
| **Validation Planner** | Experimental | converts predictions into testable plans |
| **Uncertainty Layer** | Scientific control | prevents overclaiming |

---

## Visual identity

Suggested aesthetic:

- left side: ancient, fossil, amber, stone, deep time
- right side: future, cyan, green, ecological, computational
- center: DNA/time bridge
- mood: scientific, elegant, slightly cinematic
- avoid: excessive sci-fi magic or overcomplicated infographic density

---

## Voice

PALEO-FUTURA should sound:

- ambitious but cautious
- visionary but evidence-aware
- interdisciplinary
- biologically grounded
- honest about uncertainty

Avoid sounding:

- hype-driven
- deterministic
- overconfident
- black-box AI centered
- anti-experimental
