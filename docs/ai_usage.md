# AI Usage Declaration

This repository was conceptualized and drafted with assistance from AI tools, including ChatGPT/Helios.

AI assistance may be used for:

- brainstorming project architecture
- drafting documentation
- generating code scaffolds
- suggesting test cases
- explaining algorithms
- preparing example workflows

Human review is required for:

- scientific claims
- biological interpretation
- code correctness
- statistical validity
- tool selection
- publication-ready text
- repository release decisions

AI-generated content should not be treated as automatically correct.

All analyses must be validated using appropriate scientific, statistical, and computational checks.
