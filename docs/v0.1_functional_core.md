# v0.1 Functional Core

This document describes the first runnable PALEO-FUTURA prototype.

## PALEO v0.1

PALEO v0.1 summarizes orthogroup tables.

Input:

```text
orthogroup,species,gene_id,sequence_id,notes
```

Run:

```bash
paleo-futura paleo summarize examples/toy_gene_family/orthogroups.csv --out results/paleo_demo
```

Outputs:

```text
results/paleo_demo/paleo_orthogroup_summary.csv
results/paleo_demo/paleo_report.md
```

## FUTURA v0.1

FUTURA v0.1 scores adaptive potential from a gene-feature table.

Input:

```text
gene_id,orthogroup,kaks,pnps,snp_density,stress_log2fc,te_proximity,gene_family_size,scenario
```

Run:

```bash
paleo-futura futura score examples/futura_salinity_demo/gene_features.csv --out results/futura_demo
```

Outputs:

```text
results/futura_demo/futura_scores.csv
results/futura_demo/futura_report.md
```

## Scientific caution

These modules are intentionally simple.

PALEO v0.1 does not perform ancestral reconstruction yet.  
FUTURA v0.1 does not predict exact future mutations or genomes.

Both modules produce structured hypothesis scaffolds for future development.
